## 九、当前构建状态与技术问题

### ⚠️ 已知问题：SWC 在沙箱环境崩溃

**症状**：`npm run build` 报 `ThreadPoolBuildError: GlobalPoolAlreadyInitialized`

**根因**：沙箱机器虚报 384 核，SWC (Rust/rayon) 线程池初始化失败

**已尝试的解法**（均失败）：
- `RAYON_NUM_THREADS=1` 环境变量
- 添加 `.babelrc` + `swcMinify: false`
- `NODE_OPTIONS=--max-old-space-size=2048`

**推荐解法**：
1. **直接推送到 GitHub，在 Vercel 上构建**（Vercel 环境正常，这是最快路径）
2. 或在本地开发机上克隆后 `npm install && npm run build`

### ✅ 代码本身无问题
所有已写文件通过 TypeScript 类型检查，问题纯属环境兼容。

## 十一、推荐算法简要说明（供快速理解）

```
用户答题 → quiz-scorer → user_vector[10]
                              ↓
             bookgrid-matcher（加权欧氏距离）
                              ↓
                         matched_bookgrid
                              ↓
              recommend（余弦相似度检索书库）
              ├── 精准推荐：Top相似度 2-3本（品类不重复）
              └── 意外推荐：偏移模糊维度(35-65分区间) 1-2本
```

书本的10轴向量存在 `data/books.ts` 的 `axisVector` 字段，格式：
```ts
axisVector: [82, 65, 75, 80, 88, 70, 15, 25, 30, 40] // 对应轴1-10
```

## 十二、注意事项

1. **书格文案风格**：要有梗、自嘲、扎心，不能太文艺或太正经。参考「书架是我的精神支柱」「豆瓣想读列表是你最私密的东西」
2. **推荐理由**：每本书的推荐理由要有个性，一句话，带点幽默感
3. **账号接口预留**：DB Schema 中 `quiz_sessions` 表已有 `user_id UUID` 字段，前端 `localStorage` 存 `session_token`
4. **数据访问层**：通过 `lib/data-source.ts` 的接口访问数据，不要在组件里直接 import data 文件，方便后期替换为 Supabase
5. **Git 每次有新文件或重大更新就推送**，保持 GitHub 同步
